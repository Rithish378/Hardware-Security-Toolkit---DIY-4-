# Bluetooth Device Discovery Monitor

An ESP32-based educational project for discovering nearby Bluetooth devices and monitoring wireless environments.

## Features

- Bluetooth device discovery
- Device inventory tracking
- RSSI monitoring
- Timestamp logging
- Embedded systems learning

## Hardware

- ESP32 DevKit
- USB Cable

## Project Structure

- src/ - source code
- docs/ - documentation
- screenshots/ - demo images

## Disclaimer

For educational, research, and authorized testing purposes only.
