# RFID Security Research Platform

## Overview

This project demonstrates the use of an Arduino Uno and RC522 RFID reader to study RFID-based identification systems in a controlled and authorized environment.

## Hardware

- Arduino Uno
- RC522 RFID Reader
- RFID Cards / Keyfobs
- Jumper Wires
- USB Cable

## Wiring

| RC522 | Arduino Uno |
|--------|------------|
| SDA | D10 |
| SCK | D13 |
| MOSI | D11 |
| MISO | D12 |
| RST | D9 |
| GND | GND |
| 3.3V | 3.3V |

## Project Goals

- Learn SPI communication
- Understand RFID authentication concepts
- Explore embedded systems development
- Build practical hardware security skills

This project is intended for educational, research, and authorized testing purposes only. Use only on systems, devices, and RFID tags that you own or have permission to assess.
