# Hardware-Security-Toolkit---DIY-4-
The motto of this Repo is to showcase the use of 4 Hardware Penetration Testing Tools, which can be assembled and integrated at home as a replacement for expensive tools such as #Rubberducky #proxmark #wifipinapple & more.
